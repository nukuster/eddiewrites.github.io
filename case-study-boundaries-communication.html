Boundaries & Communication Toolkit

Experimental modular language system for conflict, alignment, and emotional regulation

Role
Conversation Designer ¬∑ UX Writer
Duration
Iterative pilot
Scope
Modular language toolkit for high-stakes conversations

Context

People often struggle to communicate boundaries and needs in moments of tension, while existing advice tends to be either too abstract to use in real time or overly clinical, reducing authenticity and agency. This project focused on designing a reusable language system to support emotionally charged conversations across professional, interpersonal, and high-stakes contexts.

Constraints

  • Language needed to work across multiple relationship contexts
  • Phrases had to reduce emotional charge without sounding passive or scripted
  • Users needed flexible options rather than prescriptive scripts
  • The system had to support real-time use under stress

Approach

I approached this project as a conversation and interaction design problem, focusing on affordances rather than advice.

Identify Breakdown Moments

  • Analyzed common conflict patterns where conversations escalated or stalled
  • Identified recurring needs: pausing, redirecting, clarifying, and realigning

Design Modular Language Patterns

  • Built a phrase system inspired by conversation design and de-escalation principles
  • Defined five core ‚Äúsoft power‚Äù communication moves:
    • Pause
    • Reframe
    • Redirect
    • Clarify
    • Reset Alignment
  • Wrote microcopy patterns that preserve agency while lowering emotional intensity

Make the System Learnable

  • Documented each pattern with:
    • What this does
    • When to use it
    • How it typically feels to the other person

Design Intent

Qualitative

  • Increased user confidence in boundary-setting
  • Reduced escalation in emotionally charged conversations
  • Faster recognition of which communication move to apply
  • Language adopted flexibly rather than memorized verbatim

What This Demonstrates

Well-designed language patterns function like interaction affordances: they guide behavior without removing autonomy. This project demonstrates how conversational systems can support regulation, clarity, and agency under real-world emotional constraints.
